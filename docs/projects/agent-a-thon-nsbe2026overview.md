## About the Event

A full-day hands-on experience combining guided learning with a live Agent Sprint, where participants collaborate in teams to design and build AI agents that solve real-world scenarios using Microsoft Copilot Studio.

### Who It's For

- Solution Architects
- Developers
- Functional Consultants
- Business Technologists
- College students interested in AI, automation, and agent development

No prior Copilot Studio experience required.

### What You Learn

- Concepts and vision behind Copilot Studio
- Overview of Copilot Studio capabilities
- Creating a basic copilot and exploring topics
- Building copilots from website data
- Uploading and managing documents as knowledge sources
- Working with entities in Copilot Studio
- Creating custom actions in Copilot Studio
- Calling actions using plugins
- Publishing copilots to Microsoft Teams and websites
- Monitoring and analyzing copilot performance with analytics

### Agent Sprint & Prizes

- Teams design and build an AI agent during a live sprint
- Solutions evaluated on creativity, functionality, and impact
- 1st place team awarded prizes
- All participants earn an Agent-A-Thon Completion Badge (addable to LinkedIn)
